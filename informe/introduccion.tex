\section{Introducción}
El objetivo de este trabajo práctico es profundizar en el estudio de las Redes Definidas por Software (SDN) y el funcionamiento del protocolo OpenFlow.

El desarrollo se centra en la construcción de una arquitectura flexible capaz de soportar una topología de red dinámica, y un módulo de firewall de capa de enlace para el filtrado de tráfico.

Para la realización de estas tareas se emplea Mininet como entorno de emulación de la infraestructura de red, junto con el controlador POX, el cual opera como el cerebro centralizado responsable de la lógica de reenvío y la aplicación de las políticas de seguridad.
